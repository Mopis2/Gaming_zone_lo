Transitions animés

On va pouvoir créer diverses transitions sur des éléments pour agrémenter vos pages web.Les transitions sont très utiles pour une belle mise en page ou pour rajouter du cachet a ton site.

On ajoutera se code pour exécuter du bootstrap que j'utilise dans mon html:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@4.5.3/dist/css/bootstrap.min.css" 
integrity="sha384-TX8t27EcRE3e/ihU7zmQxVncDAy5uIKz4rEkgIXeMed4M0jlfIDPvg6uqKI2xXr2" crossorigin="anonymous">

Fondu

Je t'ai mis le code de chaques transitions avec leur application juste en dessous pour que tu puisse t'y retrouver tranquillement

On commence par un simple fondu (C'est une transition qu'on utilise assez souvent pour une page web)

<h2>Fondu<h2>
<div class="img-circle fondu"></div>
.fondu {
background-color: #3d3d3e;
height: 200px;
width: 200px;
border-radius: 50%;
transition: all 0.5s ease-out;
}
.fondu:hover {
background-color: #FF3232;
}

Pour faire ma transition tu remarquera que j'utilise hover dans mon css. Hover sert a effectuer la transition quand tu survole l'élément avec ton curseur. Mais il en existe d'autres comme link, active ou encore focus.

Résultat

Rotation

Ici, on va faire une rotation d'élément, mais je me suis permis d'ajouter le fondu de couleur fait juste au dessus pour rendre sa plus sympa.

<h2>Rotation<h2>
<div class="rotation"></div>
.rotation {
transition: all 0.5s;
height: 200px;
width: 200px;
background-color: #3d3d3e;
}
.rotation:hover {
transform: rotate (45deg);
background-color: #FF3232;
}

Dans le CSS j'utilise un transform: rotate(45deg) mais tu peux changer cette valeur selon tes envie pour faire bouger ta transition. J'aurais aussi pu ajouter des nouvelles valeurs pour la hauteur et la largeur dans mon rotation:hover ce qui aurait pour effet d'aussi changer la taille de mon élément quand je passe ma souris dessus.

Opacité

Ici, on va simplement changer l'opacité d'un élément.

<h2>Opacité<h2>
<div class="opacité"></div>
.opacité {
transition: all 0.5s ease-out;
border-radius: 50%;
opacity: 0.5;
height: 200px;
width: 200px;
background-color: #3d3d3e;
}
.opacité:hover {
opacity: 1 ;
}

Tu pourra changer l'opacité à ta guise avec le paramètre opacity:

Résultat

Délai

Ici, on va mettre un délai à un élément.

<h2>Délai<h2>
<div class="délai"></div>
.délai {
transition: background-color 0.5s ease-out 1s;
border-radius: 50%;
opacity: 0.5;
height: 200px;
width: 200px;
background-color: #3d3d3e;
}
.délai:hover {
background-color: #FF3232;
}

Si tu regarde bien, mettre un délai n'est pas très compliqué. La seul chose à ajouter c'est un temps au ease-out dans le style css.

Résultat

Zoom

Ici, on va pouvoir zoomer sur un élément.

<h2>Zoom<h2>
<div class="zoom"></div>
.zoom {
transition: transform 0.5s ease-out;
border-radius: 50%;
height: 200px;
width: 200px;
background-color: #FF3232;
}
.zoom:hover {
transform: scale(1.2);
}

Faire un zoom n'est pas très compliqué non plus. On utilise transform dans le style css, et on lui met une valeur. Moi j'ai mis 1.2 pour augmenter la taille de mon élément 1.2x.

Résultat

Déformation

Ici, on va pouvoir zoomer sur un élément.

<h2>Déformation<h2>
<div class="déformation"></div>
.déformation {
transition: transform 0.5s ease-out;
border-radius: 50%;
height: 200px;
width: 200px;
background-color: #FF3232;
}
.déformation:hover {
transform: skew(45deg);
}

Pour la déformation on utilise les mêmes lignes de code que pour le zoom, à la seule différence qu'au lieux d'utiliserscale on utilise skew dans le style css.

Résultat

Taille

Ici, on va pouvoir ajouter un effet de taille sur les axes x ou y.

<h2>Taille<h2>
<div class="hauteur"></div>
<div class="largeur"></div>
.hauteur {
transition: height 0.5s ease-out;
border-radius: 50%;
height: 200px;
width: 200px;
background-color: #FF3232;
}
.hauteur:hover {
height: 300px;
}
.largeur {
transition: weight 0.5s ease-out;
border-radius: 50%;
height: 200px;
width: 200px;
background-color: #FF3232;
}
.largeur:hover {
width: 300px;
}

Pour la taille on utilise les mêmes lignes de code que pour la déformation, à la seule différence qu'au lieux de déformer l'élément avec des angles skew on utilise height et weight pour modifier uniquemant la hauteur et la largeur.

Résultat

Déplacement

Ici, on va pouvoir déplacer un élément sur le plans x ou y.

<h2>Déplacement<h2>
<div class="déplacement"></div>
.déplacement {
transition: all 0.5s ease-out;
top: 0;
border-radius: 50%;
height: 200px;
width: 200px;
background-color: #FF3232;
position: relative;
}
.déplacement:hover {
top: 30px;
}

Le déplacement a besoin d'avoir un point de départ pour pouvoir se déplacer. Moi je fais partir mon déplacement du haut de mon élément avec top mais j'aurais pu utiliser bottom, right ou left. Ensuite je lui demande de se déplacer avec un nombre de pixels top: 30px. Il ne faut pas oublier la ligne position: relative car sans cette ligne ton élément ne bougera pas (je me suis déja fait avoir plusieurs fois).

Résultat

Voilà, avec ça je pense que tu en à assez pour te sustenter et faire de belles transitions.
